🤖 AI Hub | Smart Web Tools

Welcome to AI Hub, a web-based platform featuring powerful AI-driven tools designed for productivity and file management.

✨ Key Features
PDF Converter: Seamlessly convert documents to and from PDF format.
Text Summarizer:Get quick summaries of long articles and text using AI.
File Cleaner: Automatically clean and organize messy data files.
Responsive UI: Fully optimized for mobile and desktop views.

 🛠️ Built With
 Frontend: HTML5, CSS3, JavaScript (ES6+)
Styling: Tailwind CSS / Bootstrap
Logic: Custom JS & AI API integrations

🚀 How to Run
1. Clone the repo: `git clone https://github.com/hasnain-tech/AI-Hub.git`
2. Open `index.html` in your browser.
